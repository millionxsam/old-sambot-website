Frequently Asked Questions

Why is SamBot offline
If SamBot is ever offline, the developers may be releasing a new update which we have to reboot the bot for. If the bot is offline for a long time, please check our server to see if there is a downtime planned or report it.
How do I enable NSFW commands
NSFW commands are only available in NSFW channels. Go to channel settings to enable NSFW channel option
How do I turn on/off ticketing
You can do the command '-ticket-toggle [on/off]' to turn on or off ticket commands
How do I enable/disable automod
There should be a command on the help menu for that automod feature, for example: '-antilink on' would turn on antilink
Why isn't the SamBot website link working
For some reason, because the SamBot website is hosted on GitHub, you can't use it with 'www.' https://sambot-bot.github.io would work, but https://www.sambot-bot.github.io/sambot wouldn't work. We are trying to get a better hosting platform for the website
How do I become a SamBot developer
You can join SamBot Community Discord server and go to #bot-developer-app and fill out the google form
