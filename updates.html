Changelog V3.0

This update is so big that I can't really put all of the commands into one announcement... but you can do -help to see the new changes! Brief Overview: Moderation, Info, Utility, Ticket, Many Nsfw and Fun Commands (50+), Admin and Auto-Mod and way much more! Ticket commands are like -ticket-new, and it would create a private channel to help you with issues (Way more ticket features too). NSFW send weird pictures of inappropriate stuff (can only be used in NSFW channels). There aren't that many auto-mod features right now but more may be coming later. Now that SamBot maintenance is finally over, I'm going to focus more on making SamBot more popular and in more servers... because right now SamBot's users are mostly just my friends / friends of friends.
